[はじめに]
　本ソフトウェアは,拓殖大学の外部公開イベントの1つである卒業研究.開発にて私(大学生)が作成したものである．本ソフトウェアは独自OSのソースコード(ファーム,ドライバも含む)です．ターゲット情報などの各種情報は,下記のドキュメントを参考にして下さい．


[概要]
  本研究の目的は，OSの各機能について，複数のポリシ/メカニズムを搭載した学習向け独自OSをフルスクラッチで自作することである．
搭載する複数のポリシ/メカニズムは，基本的なOS機能であるタスクスケジューリングやメモリ管理，及びデッドロック問題や優先度逆転問題を回避する戦略を対象とする．さらに，搭載した複数のポリシ/メカニズムを，静的及び動的に切換えられる機構を用意した．また,動作の学習を可能とするサンプルタスクセットを用意した．
 これにより，学習者は本OSを使用し，複数のポリシ/メカニズムの動作を単一環境下から学ぶ事が可能である．
